# 📊 Pandas Data Merging and Lookup Operations

A structured Python project demonstrating data cleaning, join operations, and lookup automation using Pandas.  
This project was developed as part of an academic assignment to showcase practical DataFrame handling skills.

---

## 📌 Project Overview

This project is divided into three main tasks:

🔹 **Task 1: Data Preparation**  
• Created a Students DataFrame  
• Identified and analyzed null values  
• Removed invalid records  
• Replaced missing city values with "Unknown"  
• Produced a cleaned dataset ready for analysis  

🔹 **Task 2: Join Operations**  
Performed different types of joins between Students and Enrollments DataFrames:

• ✅ Inner Join  
• ✅ Left Join  
• ✅ Right Join  
• ✅ Outer Join  

Each join includes row count verification and structured output preview.

🔹 **Task 3: Lookup and Automation**  
• Performed lookup using merge()  
• Handled missing exam scores correctly  
• Created an automation function for dynamic joins  
• Tested function with multiple join types
